⚛️ Redux Toolkit (Auth)

Redux is strictly reserved for global, highly-accessed Client State (primarily Authentication variables like the JWT token, `isAuthenticated`, user role, and ID). We avoid stuffing API outputs into Redux, eliminating the "boilerplate nightmare" of THUNKs completely.

♻️ React Query (Server State)

Query handles 95% of state. Any data inherently "owned" by the backend (Mentors, Sessions, Notifications) is managed via `useQuery`/`useMutation`. It caches data automatically, dedupes parallel requests, polls dependencies continuously, and updates the View seamlessly without manual state tracking (`isLoading`, `isError`).

🚦 Route Guards

Specific application pages map strictly to Java backend JWT Role privileges. Unauthorized traversals trigger automatic router kickbacks redirecting unauthorized visitors.

  • ProtectedRoute (Base API)

    Requires user context state `isAuthenticated == true`. Protects Dashboards, Bookings, Notifications.

  • MentorRoute

    Stops `LEARNER` roles. Allows access to Mentor availability schedules, earnings overviews.

  • AdminRoute

    Only accessible via `ROLE_ADMIN` context token. Renders the specialized analytics and Mentor Approval tables.
